UxDIL Software Collection

Templates

This folder contains folders with files, which you can use as template for your project. The following templates are available:

  • Serial:
    Communication Device Class (CDC)
    The USB device is a virtual serial interface

  • RawHID:
    Human Interface Device (Raw HID)
    A USB device with standardized data exchange. No special drivers required.

  • UserDevice:
    A user defined USB device.
    Needs own drivers, but gives greatest flexibility.
